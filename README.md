# Speech-To-Text-Recognition-System
The purpose of this repository is to implement the TRAINING OF A SPEECH RECOGNTION SYSTEM USING SYLLABLES in Java.
The API adopted is the Sphinx4 Speech API while Java SWING is used for designing of the User Interface.
Get the full API from https://sourceforge.net/projects/cmusphinx/files/sphinx4/ and include the necessary jar files in your project's library
